Problem

Marketing and service teams shipped separate experiences, so storytelling broke once members clicked “book.”

Goal

Blend storytelling + service flows so recruiters and stakeholders see brand craft plus production rigor.

Approach

Map onboarding flows → define gradients + photography → prototype modules → document `/blueprint` + docs.

Outcome

Live site + blueprint + docs recruiters can inspect; engineers reuse tokens + accessibility guidelines.

Brief

Design a unified marketing + service layer for a premium fitness brand with emphasis on schedule clarity, brand gradients, and onboarding modules.
