Case Study · Marketing + Service

Horizon Fitness Experience

Crafted a hybrid fitness marketing + service experience with module-based schedule UI, typography, gradients, and `/blueprint` walkthroughs that showed how to scale across devices. The system improved trial-to-member conversion by grounding the story in flows.

View live on Vercel Blueprint route Docs folder GitHub repository Figma system

Brief

Design a unified marketing + service layer for a premium fitness brand with emphasis on schedule clarity, brand gradients, and onboarding modules.

Role

  • Solo product + marketing designer
  • Content strategy + flow mapping
  • Design system + motion specs

Deliverables

  • Marketing page + interactive modules
  • `/blueprint` annotated flow
  • Docs describing tokens + gradients

Highlights

  • Documented responsive grids and typography scales that kept marketing + logged-in surfaces aligned.
  • Schedule module demonstrates multi-state components (idle, hover, booked) with tone + accessibility cues.
  • Blueprint page mirrors the Figma walkthrough with gradients, imagery rules, and animation timings.

Review Checklist

  • Scan the live site for storytelling, imagery, and CTA rhythm.
  • Jump to `/blueprint` for IA, responsive specs, and developer notes.
  • Open docs + Figma to see tokens, typography, and gradient usage guidelines.
